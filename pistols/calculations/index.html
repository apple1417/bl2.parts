BL2/TPS Pistol Calculation Guide

This guide covers how exactly to calculate pistol stats, and assumes you've already read the base guide here.

There are two important concepts that are going to be used here a lot:

Firstly, the general formula for all stat bonuses is as follows:
FINAL = (BASE + PreAdd) × (1 + Positive Scale)÷(1 - Negative Scale) + PostAdd
All bonuses in a specific category are added together before being applied.

The other big concept is grades ("g" for short). Grades are simply an easy way to have multiple ranks of bonuses for different parts. A stat bonus will be defined by its base value and a per-grade bonus.
BASE + PERGRADE×Grades
All grades from all parts are added up before being applied. This matters for scale bonuses, where a positive base and negative per-grade value will eventually change the scale to a negative one, or vice versa.
Note that the previous page lied a bit, inverting some bonuses to make all of that type have the same direction. A positive grade bonus may well decrease the raw value. The colour of each bonus (if it's good or bad) is still accurate however.

Weapon Type Definition

Weapon Type Definitions are the most important part when it comes to calculating stats. They define the formulas for the base values for each stat, as well as the base and per-grade bonuses. This means even having the exact same parts on weapons with two different definitions can give you different stats.
There are 8 different definitions, one for each manufacturer. This table shows how each definition defines their base and per-grade stats. There are more attributes which use grades, but their formulas are not fully known.

Element

All parts except the non-elemental one give a penatly of -0.2 damage.
There is one unique element part, used for the mission weapon in Hot Head, which is a copy of the standard cryo element part with all the same stats. The only difference is that it has slightly more blue lights and that it uses different prefixes. This means occasionally this gun will use the prefix "Chill", rather than the standard "Frozen" for a maliwan pistol, but is otherwise identical.
