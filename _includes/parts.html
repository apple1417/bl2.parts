{%- comment -%} Displays a part block, with all their relevant bonuses. Args: parts: An array of part entries to display. meta: The relevant weapon metadata. id: An id to give the surrounding div. If not defined, no id is given. uniques: If unique parts from the array should be included. mesh_image: If to display an image of the part's mesh. image_class: A string of custom classes to apply to mesh images. mesh_name: If to display the name of the part's mesh. hide_bonuses: If to hide the bonuses block. overrides: A list of overrides. See below. Passthrough args to `bonus.html`: simple_bonuses: Display simple bonuses, which calculate the exact bonus value and only display good/bad instead of proper bonus type. Overrides ========= Some manual adjustments might need to be made to the layout this generates - most commonly adding footnotes to oddities. This can be done using the `overrides` arg.
