The formula for all stat bonuses except status effect chance is as follows:
FINAL = (BASE + PreAdd)×SCALEVALUE + PostAdd
If Scale is positive: SCALEVALUE = 1 + Scale
If Scale is negative: SCALEVALUE = 1 ÷ (1 + Scale)
All scale bonuses from a single part are added together, scale bonuses from seperate parts are applied seperately.
All grades from all parts are added up before being applied as a single bonus.

The base grenade damage is found using one of the following formulas:
BL2TPS
100×(1.13^level) 100×(1.10^level)
Status effect chance bonuses directly multiply the base status effect chance, which depends on the element and enemy type.
