The general formula for all stat bonuses is as follows:
FINAL = (BASE + PreAdd) × (1 + Positive Scale)÷(1 - Negative Scale) + PostAdd
All bonuses in a specific category are added together before being applied.
All grades from all parts are added up before being applied as a single bonus.

The base grenade damage is found using one of the following formulas:
BL2TPS
100×(1.13^level) 100×(1.10^level)
The base status effect chance is 1. The final value is then multipled by one of the following values, based on element and enemy type.
